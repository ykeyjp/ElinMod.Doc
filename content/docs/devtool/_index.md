## YK DevToolとは？

[YK Testing Tool](https://steamcommunity.com/sharedfiles/filedetails/?id=3365678112)の後継として開発されたModです。  

`YK Testing Tool`自体もMod開発における実験的Modでした。  
それらで得たノウハウをもとに、[YK Framework](../framework/) と `YK DevTool`を開発しました。
